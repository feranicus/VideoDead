/ Safe & in your control

You are always in charge

🔑

Only you get in

A private password protects the page. You can add a one-time phone code for extra safety. No shared default password ever.

🛡️

It refuses tricks

The system blocks sneaky links that try to reach private or hidden addresses. It only follows real public video links.

🧹

It forgets your files

Downloaded videos are wiped from the server after a few hours, so nothing piles up and nothing lingers.

🔒

Locked connection

Everything travels over an encrypted (HTTPS) connection — the same lock your bank uses.

⚖️

Lawful use

For videos you own or may download. It does not break paid services' protection (like Netflix). You stay on the right side.

🧰

Built on a proven engine

The downloading is done by yt-dlp, a trusted open-source tool used by millions — we just wrapped it in a friendly button.
